## Key Design Decisions

| Concern | Choice | Rationale |
|---------|--------|-----------|
| **Async ORM** | SQLAlchemy 2.0 + asyncpg | Non-blocking DB I/O scales better with FastAPI |
| **Task queue** | Celery + Redis broker | Battle-tested; pub/sub doubles as live notification bus |
| **Real-time** | WebSocket + Redis pub/sub | Zero polling; multi-worker safe (Redis fans out events) |
| **Storage abstraction** | `local` \| `s3` switch | Same code, different env вҖ” easy local dev, scalable prod |
| **Migrations** | Alembic with `postgresql.ENUM` | Avoids enum duplicate-create bug with async asyncpg |
| **Auth tokens** | JWT access (15 min) + refresh (30 d, rotating) | Short-lived access reduces blast radius; refresh enables silent renewal |
| **Email tokens** | SHA-256 hashed, single-use | Token in DB is never the plaintext sent to user |
| **Anti-enumeration** | `forgot-password` always returns 200 | Prevents attackers from probing which emails exist |
| **Rate limiting** | Per-IP token bucket (Redis-backed) | Survives multi-worker setup; auth endpoints get stricter limit |
| **Error tracking** | Sentry (backend + frontend) | One dashboard for FastAPI tracebacks + React component errors |

## Service Responsibilities

### `api` (FastAPI)
- HTTP REST + WebSocket endpoints
- Auth, file upload validation, prediction orchestration
- Publishes Celery tasks (does NOT execute AI calls itself)
- Subscribes to Redis pub/sub to forward updates to connected WebSocket clients

### `celery-worker`
- Listens on `predictions` queue
- Executes `run_prediction`: read file вҶ’ call AI вҶ’ save result вҶ’ publish update
- Retries up to 3Г— with exponential backoff on `AIServiceError`

### `ai-model` (Flask)
- Stateless inference service
- Loads CheXNet weights once at startup
- Single `POST /` endpoint accepting image bytes

### `postgres`
- Single source of truth for users, predictions, tokens, audit logs
- All migrations versioned via Alembic

### `redis`
- Celery broker (`db=0`)
- Result backend (`db=1`)
- Rate limiting + WebSocket pub/sub channels
